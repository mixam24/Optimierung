Описание взаимодействия с репозиторием:

1. Регистрируемся в GitHub
2. Качаем клиент для своей операционной системы 
	Windows:	https://windows.github.com/
3. Авторизируемся в клиенте
4. Авторизируемся на сайте GitHub и перезаходим в этот репозиторий
5. Жмем кнопку "Clone in Desktop" внизу правой панели сайта
6. Вуаля, все готово. По остальным вопросам обращаться в других местах.
