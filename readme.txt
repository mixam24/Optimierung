Описание взаимодействия с репозиторием:

1. Регистрируемся в GitHub
2. Качаем клиент для своей операционной системы 
	Windows:	https://windows.github.com/
3. Авторизируемся в клиенте
4. Авторизируемся на сайте GitHub и перезаходим в этот репозиторий
5. Жмем кнопку "Clone in Desktop" внизу правой панели сайта
6. Далее внутри клиента создаем новую ветку:
	Находим вверху надпись master, жмем на треугольник рядом
	Нажимаем Manage
	Нажимаем на знак + для master
	Вводим произвольное имя для своей ветки.
	Жмем на только что созданную ветку, чтобы эта ветка стала текущей веткой.
7. Теперь перетаскиваем мышкой новую созданную ветку и master ветку на прямоугольники внизу,
   для того, чтобы скопировать файлы из основной ветки в вашу собственную (слева от кнопки merge должно быть название ВАШЕЙ ветки)
8. Нажимаем merge.
9. Вуаля, все готово. По остальным вопросам обращаться в других местах.
